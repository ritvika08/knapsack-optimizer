# 🎒 Knapsack Optimizer Web App

## 📌 Project Description
This project is a web-based application that demonstrates the use of **Greedy Algorithm** and **Dynamic Programming (DP)** using the Knapsack problem.

## ⚙️ Features
- User can enter item values and weights
- Calculates best selection using:
  - Greedy Approach
  - Dynamic Programming
- Displays optimal result comparison

## 🧠 Algorithms Used
### Greedy Algorithm
Selects items based on value-to-weight ratio (fast but not always optimal)

### Dynamic Programming
Finds the optimal solution using table-based approach
